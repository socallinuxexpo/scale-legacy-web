Open Source Software In Education (OSSIE)

There are many opportunities to use Open Source software in education. Come see what's available! The 3rd Annual Open Source Software in Education conference will be held on Friday, February 19, 2010 at the Los Angeles Airport Westin.

To attend OSSIE, the Open Source Software in Education conference please register on our website or in person.

 

Students and Teachers attend Friday's sessions for $10.

For $30 (50% off full adminssion!), students and teachers can attend Friday, Saturday and Sunday by using the STDNT discount code when purchasing your ticket. Be sure to bring valid school ID with you during SCALE check-in.
