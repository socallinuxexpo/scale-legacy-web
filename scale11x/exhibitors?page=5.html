OpenVZ
OpenVZ

The OpenVZ project is container virtualization for Linux, implemented in the Linux kernel. This allows a user to have multiple isolated Linux instances on a single machine. OpenVZ has many features, such as advanced resource management and live migration. The OpenVZ project maintains the kernels and the userspace tools for the project. In addition the developers work toward integrating pieces of the container functionality into the mainstream Linux kernel.
