CFP Information

The Linux Exposition of Southern California is proud to announce the 10th annual Southern California Linux Expo -- SCALE 10x -- will be held January 20-22, 2012 at the Hilton Los Angeles Airport Hotel.

To celebrate the 10th anniversary of SCALE, we will be accepting proposals for papers in these tracks:

(Note: 17 Nov, 2011 is the deadline for abstract and proposal submissions)

  • System Administration

  • Kernel Developer

  • Application Developer

  • Cloud/Virtualization

  • General Open Source / Free Software Advocacy and Technical Content

Topics of Interest

Topics of interest which have been subjects of presentations in the past include, but are not limited to:

  • Kernel Internals and Enhancements

  • Unix variants: Tools and Appliances

  • Open Source for Embedded Platforms

  • Open Source development tools for Mobile Devices

  • Open Source for Cloud Computing

  • Open Source for Unified Communications

  • In-depth Programming/Scripting with Open Source Languages (Examples include Perl, Ruby, PHP, Python, etc.)

  • Desktop Operating Systems

  • Linux/Unix/Windows Inter-operability

  • Shell Programming

  • Open Source Productivity Applications

  • Tools for Multimedia and Gaming

  • Tools for Profiling and Performance Tuning

  • Open Source Animation Tools

  • Open Source Database Platforms

  • Open Source Licensing

  • Government Policies with Open Source

  • Open Source Promotion and Adoption: Current State

  • Open Source Success Stories

  • Open Source Audio/Video Manipulation tools

Presenters may have an opportunity to host tutorial sessions in a separate track. If you believe your presentation is appropriate for a hands-on lab, please mention that in the proposal. The reviewers will determine if the content is suitable for a hands-on session.

How to Submit a Proposal

  1. Visit the Speaker Registration page and register by filling in the required fields. Registrations from previous years are no longer valid. Please ensure that the personal biography is not more than 1,000 characters; focus on relevant experience to your proposal topic and write from the third-person perspective. Note that the upper email-address field is the one that will be used to confirm the speaker registration.

  2. Once registration is complete, an email will be sent confirming the speaker registration and a link to activate the account.

  3. After returning as a registered speaker, click on the Submit presentation link. You may submit multiple presentations and you may include co-speakers who are already registered as speakers.

If you have problems during the submission process, please email info@socallinuxexpo.org.

SCALE provides the presenters with:

  • microphone

  • overhead projector

  • a laptop running Linux with a PDF viewer as well as LibreOffice or OpenOffice.

If you require special equipment, contact us at info@socallinuxexpo.org.

Presenters will be allotted 45 minutes for their presentations. Please ensure that presentations are in an open, non-proprietary format.

The number of quality submissions received has increased substantially every year. Proposals are reviewed by an expert committee and evaluated solely on their merits. We request that submission dates be strictly honored in order to provide the committee enough time to choose the best set of proposals.
