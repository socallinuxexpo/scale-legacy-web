Cloud Computing and Open Source - the eyeOS Project
Title: 
eyeOS

Abstract: 

*Cloud Computing* is hitting this days the technology books and newspapers. It's all about working and doing almost everything from Internet. But usually this model means giving away our privacy.

The eyeOS Project was born to create a /Cloud Computing Operating System/, hosted from any server (recommending a Linux machine to act as server) and allowing anyone to access their data and work with their files from everywhere in the world - just with a web browser. Currently eyeOS is available under 32 languages and has an active community in 9 different countries.

More information about eyeOS can be found at http://www.eyeos.org.
