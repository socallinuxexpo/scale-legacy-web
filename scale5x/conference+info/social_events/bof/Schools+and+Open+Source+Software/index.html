Schools and Open Source Software
Host: Tim Fritchtel - Schools and Open Source Software
Topic:

Are you using or interested in using Open Source Software in your school? If so, join us to share your experiences and learn about some of the great free software available to schools.

We'll start with a whirlwind tour of software ranging from standard tools like Open Office (Office Suite) and Firefox (web browser), to creative tools like GIMP (photo editing), Inkscape (drawing), Scribus (desktop publishing) and N|Vu (web publishing).

We'll also take a quick look at powerful infrastructure tools like LTSP and K12LTSP (thin client computing), Moodle (Learning Management System), Wordpress (blogging) and Joomla (web publishing). Time permitting, we'll review how Open Source Software is being used in schools across the nation.
