Open Source Software in Education

There are many opportunities to use Open Source software in education. Come see what's available! The SCALE Education conference will be held on Friday, February 8th, prior to the 6th Annual So Cal Linux Expo.

If you've already deployed Open Source software in education, please share your experiences with the attendees.
