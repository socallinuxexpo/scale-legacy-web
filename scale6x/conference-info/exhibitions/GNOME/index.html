Foresight Linux

Foresight Linux is a Distribution which showcases the latest and greatest version of the GNOME Desktop Environment and software from Freedesktop.org.

Foresight Desktop Linux is a GNOME-based Linux distribution that showcases some of the latest bells and whistles. It currently sports GNOME 2.20 as the preferred desktop environment. Some of the features of the Foresight Desktop include the Beagle search tool, the Epiphany web browser, F-Spot photo management application, Banshee music management, and the Conary packagement system.

The Foresight Linux team will demo the latest release of Foresight Linux.

Visit website for more information: http://www.foresightlinux.org
